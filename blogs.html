What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programs to interact with and manipulate the documentâ€™s structure, style, and content

How do You select an element from the DOM?

We can use getElementById, getElementsByClassName, getElementsByTagName, querySelector,querySelectorAll for select an element from the DOM

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that leverages event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated

How do you manipulate an element's attributes and styles using the DOM?

Get an Attribute,Set an Attribute,Remove an Attribute,Adding/Removing Classes,Toggle Classes by these we can manipulate an element's attributes and styles using the DOM
